# webdriver-sdk-csharp
A library for webdriver sdk with c#


# 介绍

本项目是webdriver sdk的c++库封装

开发环境: Visual Studio 2010

# 目录

lib - 存放sdk .lib和.dll文件

demo_csharp - 演示如何调用sdk


# 使用

1. 新建项目，打开项目属性
2. 引用lib中的webdriver_sharp.dll文件
3. 编写代码
4. 生成并运行

注意，需要把lib下的文件拷贝到项目目标目录下
